# flutter_app

Demo Flutter App

## Getting Started

 Before running the project make sure to you have flutter and its components are configured in your machine. https://flutter.dev/docs/get-started/install

 ```
    // mac Os
    git clone https://github.com/manjufy/flutter_app.git
    > cd flutter_app
    > open -a Simulator
    > flutter run
 ```
